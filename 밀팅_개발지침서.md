## 6. 구글 시트 연동 구조

```
신청자 폼 제출
    ↓
Next.js API Route (/api/signup)
    ↓ encodeURIComponent(JSON)
Google Apps Script 웹훅 (GET 요청)
    ↓
구글 시트에 행 추가 + 운영자 이메일 발송
```

### 데이터 흐름 상세
- API route에서 폼 데이터를 JSON으로 만들어 `encodeURIComponent`로 인코딩
- Apps Script URL에 `?data=인코딩된값` 형태로 GET 요청
- Apps Script에서 `decodeURIComponent` + `JSON.parse`로 파싱 후 시트에 저장

### 구글 시트
- URL: https://docs.google.com/spreadsheets/d/1UmNvYcBGKI7JMj4Rq74D04Bm_0Ra7Gqo5VnEykluvxo/edit
- 컬럼 순서: 신청일시 / 이름 / 직군 / 회사 / 빌딩 / 연락처 / 메모

## 7. Google Apps Script 코드

> Apps Script 편집기: 구글 시트 → 확장 프로그램 → Apps Script

```javascript
function doGet(e) {
  try {
    const sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
    const decoded = JSON.parse(decodeURIComponent(e.parameter.data));

    sheet.appendRow([
      new Date(),
      decoded.name     || "",
      decoded.role     || "",
      decoded.company  || "",
      decoded.building || "",
      decoded.contact  || "",
      decoded.memo     || "",
    ]);

    MailApp.sendEmail({
      to: "luckandy1234@gmail.com",
      subject: "🍱 밀팅 새 신청자: " + decoded.name,
      body: [
        "새로운 점심 신청이 들어왔습니다!",
        "",
        "이름: "   + decoded.name,
        "직군: "   + decoded.role,
        "회사: "   + decoded.company,
        "빌딩: "   + decoded.building,
        "연락처: " + decoded.contact,
        "메모: "   + decoded.memo,
        "",
        "👉 구글 시트 확인: https://docs.google.com/spreadsheets/d/1UmNvYcBGKI7JMj4Rq74D04Bm_0Ra7Gqo5VnEykluvxo/edit",
      ].join("\n"),
    });

    return ContentService
      .createTextOutput(JSON.stringify({ success: true }))
      .setMimeType(ContentService.MimeType.JSON);
  } catch (err) {
    return ContentService
      .createTextOutput(JSON.stringify({ error: err.toString() }))
      .setMimeType(ContentService.MimeType.JSON);
  }
}

function testEmail() {
  MailApp.sendEmail({
    to: "luckandy1234@gmail.com",
    subject: "🍱 밀팅 이메일 테스트",
    body: "이메일 알림이 정상 작동합니다!"
  });
}
```

### Apps Script 재배포 방법
1. 코드 수정 후 저장
2. 배포 → 배포 관리 → 연필 아이콘
3. 버전: **새 버전** 선택
4. 배포 → 새 URL 복사
5. `.env.local` 및 Vercel 환경변수 업데이트

## 11. 트러블슈팅

### Apps Script 한글 깨짐
- 원인: URL 파라미터 인코딩 문제
- 해결: `encodeURIComponent(JSON.stringify(data))` → Apps Script에서 `decodeURIComponent` 파싱

### Apps Script 이메일 권한 오류
- 원인: MailApp 권한 미승인
- 해결: Apps Script 편집기에서 `testEmail` 함수 직접 실행 → 권한 승인

### Vercel 500 에러
- 원인: 환경변수 `SHEET_WEBHOOK_URL` 값이 비어있음
- 해결: Vercel → Settings → Environment Variables → 값 직접 입력 후 Redeploy
